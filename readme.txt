Professional Blog Post Grid For Elementor

== Description ==

Postify is a professional post layout addon for elementor. The plugin will allow you to create differen types of posts layouts in your WordPress site with very simple way. You will able to create different grid layouts for your blog posts. After activating the plugin you just need to add Grid widget under 'Postify' category form your elementor widget list.

Plugin features:

* 3 Grid layout with different columns
* Use as post per page
* Can be set excerpts length
* Navigation/pagination show and hide option for carousel layout
* Loop option for carousel layout
* Show or hide option for each element
* Extensive style options.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Upload `postify-for-elementor.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Grid widget from Postify category
